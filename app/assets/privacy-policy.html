바이버스챗 개인정보처리방침

시행일: 2026년 4월 19일 | 최종 수정: 2026년 4월 19일

바이버스챗은 서버에 개인정보를 수집하지 않습니다. 모든 데이터는 사용자의 기기에만 저장됩니다.

1. 수집하는 개인정보 항목

바이버스챗(이하 "앱")은 다음 정보를 기기 내에서만 처리합니다:

  • AI API 키 — 사용자가 직접 입력한 API 키 (기기 보안 저장소에 암호화 저장)
  • 대화 기록 — AI와의 채팅 내용 (기기 로컬 저장소에 저장)
  • 음성 데이터 — 음성 입력 시 기기 내 음성 인식 엔진으로 처리 (외부 전송 없음)
  • 연결 설정 — PC 연결을 위한 IP 주소, 포트 정보 (기기에만 저장)

2. 개인정보 이용 목적

  • AI 모델과의 대화를 위한 API 인증
  • 대화 기록 저장 및 이어하기 기능
  • 음성 입력을 텍스트로 변환
  • 바이버스PC와의 터미널 연동

3. 개인정보 보관 기간

모든 데이터는 앱을 삭제하면 함께 삭제됩니다. 서버에 보관되는 데이터는 없습니다.

  • 대화 기록: 최근 50개 세션까지 기기에 보관
  • API 키: 사용자가 직접 삭제하거나 앱 삭제 시 제거

4. 제3자 제공 및 외부 서비스

사용자가 직접 API 키를 입력한 경우, 대화 내용이 해당 AI 서비스로 전송됩니다:

  • Google Gemini API — 이용약관
  • Anthropic Claude API — 이용약관
  • Groq API — 이용약관
  • OpenAI API — 이용약관
  • Microsoft Edge TTS — 텍스트 음성 변환용

각 서비스의 개인정보 처리는 해당 서비스의 약관을 따릅니다.

5. 이용자 권리

  • 삭제 권리 — 설정에서 대화 기록 및 API 키를 직접 삭제할 수 있습니다.
  • 접근 권리 — 모든 데이터가 기기에 저장되므로 언제든 확인 가능합니다.
  • 처리 거부 — API 키를 입력하지 않으면 외부 서비스로의 데이터 전송이 없습니다.
